1. O que é o Base Builder?

O Base Builder é um modo de jogo para o Counter-Strike que consiste em construtores (CTs) usarem objetos de um mapa para construir bases contra zombies (TRs).
Originalmente feito por Tirant, este modo de jogo ganhou muita popularidade devido às possibilidades sem fim que os construtores têm para construir bases únicas.
Nas palavras do próprio autor: "Pois cada usuário é diferente e vai querer construir os seus próprios estilos diferentes de bases, cada ronda é diferente. É quase impossível recriar bases exatas (a não ser que o usuário tenha praticado significativamente), assim jogar um mapa não vai se tornar muito repetitivo muito rapidamente."

2. Como é que eu jogo isto?

O Base Builder está dividido em 3 fases: a fase de construção, a fase de preparação e a fase de defesa.
Durante a fase de construção, os construtores devem construir as suas próprias bases ao olhar para um objeto e segurar a sua tecla "+use" (E por padrão). O objeto vai tornar-se numa cor pura e transparente. Enquanto o construtor está a segurar a sua tecla "+use", ele pode movê-lo ao olhar para onde ele quiser que vá.
Adicionalmente, o construtor pode usar a sua tecla "+attack" (Botão Esquerdo do Rato por padrão) e a sua tecla "+attack2" (Botão Direito do Rato por padrão) para empurrar ou puxar o objeto respetivamente. Quando o construtor estiver satisfeito com a localização do objeto, ele pode largar a sua tecla "+use" e irá tornar-se num objeto sólido.
Além disso, o construtor pode usar a sua tecla "+reload" (R por padrão) para rodar o objeto por 90 graus enquanto está a agarrar o objeto.
Ademais, o construtor pode usar a sua tecla do spray (T por padrão) para trancar/destrancar o objeto (até 15 objetos trancados), independentemente de ele estar ou não estar a agarrar o objeto, o que vai tornar o objeto numa cor pura e transparente, tornando-o imóvel para outros jogadores (com exceções).
Também, o construtor pode usar a sua tecla da lanterna (F por padrão) para copiar o objeto (até 5 cópias), independentemente de ele estar ou não estar a agarrar o objeto, exceto se o objeto estiver trancado por outro construtor (com exceções).
Finalmente, o construtor pode usar a sua tecla "+alt1" (sem padrão) para usar o gancho para uma locomoção mais rápida.
Após a fase de construção acabar, a fase de preparação vai começar e os construtores vão ser renascidos para testar as suas bases antes de os zombies serem soltos. Nesta fase, os construtores não conseguem construir mais e podem usar a sua tecla "buy" (B por padrão) para abrir um menu para comprar armas. Ao comprar as suas armas, os construtores recebem uma granada HE.
Após a fase de preparação acabar, a fase de defesa vai começar e a barreira que mantém os zombies confinados vai ser removida, permitindo que eles comecem a atacar os construtores. Nesta fase, os construtores que morrem vão renascer como um zombie e os zombies renascem ao morrer até que a ronda acabe.
Se o tempo da ronda expirar, os construtores ganham a ronda. Se os zombies matarem todos os construtores antes do tempo da ronda expirar, os zombies ganham a ronda.

3. Dinheiro

Os jogadores podem ganhar dinheiro ao matar zombies, matar humanos, morrer como zombie e passar tempo no servidor. Abaixo estão os valores de cada recompensa:

-> Matar um zombie: $4 ($6 para VIPs)
-> Matar um humano: $10 ($15 para VIPs)
-> Morrer como zombie: $2 ($3 para VIPs)
-> Passar 10 minutos no servidor: $5 ($10 para VIPs)

4. Armas

Os jogadores possuem por padrão uma SG-552 e uma USP. Porém, eles podem adquirir mais armas com dinheiro no menu de adquirir armas.

5. Melhorias

Os jogadores podem melhorar vários atributos para ambos humano e zombie. Abaixo estão os atributos que podem ser melhorados:

-> Humano:
  -> Vida
  -> Colete
  -> Velocidade
  -> Recuo
  -> Munição
  -> Dano
  -> Tempo de Recarga

-> Zombie:
  -> Vida
  -> Colete
  -> Velocidade
  -> Gravidade
  -> Coice
  -> Atordoamento
  -> Visibilidade

6. Clãs

Os jogadores podem criar um clã ao gastar uma taxa de $500. Depois de criado, o dono pode convidar até mais 5 jogadores para o clã (máximo de 6 membros por clã).
Os membros do clã podem mover/rodar/destrancar/copiar os blocos trancados por outros membros.
Os membros do clã podem ganhar dinheiro para o clã ao matar zombies, matar humanso, morrer como zombie e passar tempo no servidor. Abaixo estão os valores de cada recompensa:

-> Matar um zombie: $2 ($3 para VIPs)
-> Matar um humano: $5 ($8 para VIPs)
-> Morrer como zombie: $1 ($2 para VIPs)
-> Passar 10 minutos no servidor: $2 ($4 para VIPs)

O dono do clã pode melhorar vários atributos para ambos humano e zombie que beneficiam todos os membros do clã. Abaixo estão os atributos que podem ser melhorados:

-> Humano:
  -> Vida
  -> Colete
  -> Dano
  -> Tempo de Recarga

-> Zombie:
  -> Vida
  -> Velocidade
  -> Atordoamento
  -> Visibilidade

Só o dono do clã pode:
-> Transferir posse
-> Comprar melhorias de clã
-> Convidar jogadores
-> Expulsar membros
-> Dissolver o clã

7. Party

Os jogadores podem criar uma party com outro jogador. Depois de criado, os dois jogadores podem mover/rodar/destrancar/copiar os blocos trancados de cada um.

8. Comandos úteis

-> +use (E por padrão) para mover objetos
-> +attack (Botão Esquerdo do Rato por padrão) para empurrar objetos
-> +attack2 (Botão Direito do Rato por padrão) para puxar objetos
-> +reload (R por padrão) para rodar objetos (só enquanto estiver a agarrar um objeto)
-> impulse 201, aka spray (T por padrão) para trancar/destrancar objetos
-> impulse 100, aka lanterna (F por padrão) to copiar objetos
-> +alt1 (sem padrão) para usar o gancho para uma locomoção mais rápida
-> buy (B por padrão) para abrir o menu de compra (só disponível após a fase de construção)
-> /menu ou chooseteam (M por padrão) para abrir o menu principal
-> /unstuck ou /stuck para desprenderes-te
-> /rank para ver a tua posição no servidor
-> /top ou /top15 para ver os 15 melhores jogadores no servidor
-> /weapons ou /guns para abrir o menu de adquirir armas
-> /respawn ou /revive para renascer (só disponível antes da fase de defesa e só para construtores)
-> /upgrades para abrir o menu de melhorias
-> /clan para abrir o menu do clã (se tu não estás num clã, isto é o comando para criar um)
-> /clanaccept para aceitar um convite de clã
-> /clandecline para recusar um convite de clã
-> /checkupgrades para mostrar as tuas melhorias atuais no chat
-> /clanupgrades para mostrar as melhorias atuais to teu clã no chat
-> /party para obriar o menu de criar party
-> /partyleave para sair da tua party
-> /partyaccept para aceitar um convite de party
-> /partydecline para decusar um convite de party
-> /rules para ver as regras
-> /help para ver esta página
